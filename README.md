# Ejercicios-cuadernos-Python
Repositorio con ejercicios prácticos de fundamentos en Python, incluyendo estructuras de datos (cadenas, listas, tuplas, conjuntos y diccionarios), estructuras de control (condicionales y bucles), funciones y clases.

El objetivo es fortalecer la lógica de programación, comprender la sintaxis del lenguaje y desarrollar habilidades en la resolución de problemas mediante práctica aplicada y ejercicios progresivos.

Este repositorio funciona como evidencia de aprendizaje y como base para el desarrollo de proyectos más avanzados en Python.
